Símbolos e Emoji

Símbolos Especiais

Existem diversos caracteres diferentes que não temos teclas especificas no teclado, alguns exemplos que podem ser aplicados no código são:

© ® ™ ¥ £ ¢ ↑ € Δ α β γ

Para encontrar mais, pesquise por "misc symbols" no Google.

Emoji

Pesquise por "emojipedia.org" e procure pelo código U+ do emoji desejado. Estruture com &#x e cole o código encontrado, por exemplo:

💀 🥀 💸 💎 ‼ 💤

Bônus

Também podemos fazer comentários no código, usando "<!--" que o vs code vai completar a tag de fechamento, cujo é "-->".
