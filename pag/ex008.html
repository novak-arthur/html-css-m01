Formatação de texto

Negrito / Destaque

Nesta linha quero usar um termo em negrito (não semântico).
(tag <b>)✗

Nesta linha quero usar um termo em destaque (semântico).
(tag <strong>)✓

A tag <b> só o deixa em negrito, coisa que a tag <strong> também faz, mas dá significado, indica destaque.

Itálico / Ênfase

Nesta linha, quero usar um termo em itálico (não semântico).
(tag <i>)✗

Nesta linha, quero usar um termo em ênfase (semântico).
(tag <em>)✓

Assim como a tag <b>, a tag <i> só o deixa em itálico, já a tag <em> também, mas dá significado, indica ênfase.

O que torna o código semântico ou não, é o significado que é atribuido. De nada adianta dizer que é negrito por ser negrito, mais sentido é dado com a tag que faz o mesmo e ainda indica no seu código o destaque no texto.

Marcado

Nesta linha, quero marcar um trecho.
(tag <mark>)

Também é possível modificar a cor do trecho grifado com CSS.

Pequeno e Grande

Aqui, quero uma palavra grande e outra pequena.
(a tag <big> entrou em desuso, pois é considerado não semântica, já a tag <small>, não.)

Deletado

Quero utilizar um trecho legivel mas não considerado, como esse aqui..
(tag <del>)

Inserido

Agora, uso um trecho que parece ter sido inserido, como esse aqui também.
(tag <ins>)

Outro método é usando a tag <u>, mas não é considerada semântica.

Sobrescrito

Para termos matemáticos, como potenciação, x20+1.
(tag <sup>)

Subscrito

Já para termos químicos, como H2O.
(tag <sub>)

Código-fonte / Pré-formatação

Para que códigos se mantenham com a fonte monoespaçada, isso é, espaço iguais entre as letras, utilize a tag <code>.

Exemplo (com): document.getElementById('teste')

Exemplo (sem): document.getElementById('teste')

Agora para códigos manterem o espaçamento e a quebra de linhas, utilize <pre>. Ex.: 

        
    num =int(input('Digite um número'))
    if num % 2 == 0;
        print(f'O número {num} é PAR')
    else:
        print(f'O número {num} é ÍMPAR')
    print('fim do programa')
