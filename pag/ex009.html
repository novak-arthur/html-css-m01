Tipos de Listas

Lista Ordenada

A tag usada para listas ordenadas é <ol>. Para adicionar itens, tag <li>.

  1. Acordar
  2. Escovar os Dentes
  3. Tomar Banho
  4. Tomar Café
  5. Estudar

Lista não Ordenada

Diferente da lista ordenada, a lista não ordenada utiliza a tag <ul>.

  • Arroz
  • Cebola
  • Ovo
  • Sal
  • Azeite
  • Frango
  • Pão

Lista de Definições

A lista de definições é feita com a tag <dl>, os itens com <dt> e a descrição com <dd>

HTML
Linguagem de marcação utilizada para a criação de conteúdo de um site.
CSS
Linguagem de marcação utilizada para o design de um site.
JavaScript
Linguagem de programação utilizada para a interatividade de um site.

No HTML5, não precisa mais fechar a tag <li>, funciona do mesmo jeito.
